Structural Equation Modeling


Course website: https://enricotoffalini.github.io/SEM-course/

Short description
This course introduces the foundations of Structural Equation Modeling (SEM), covering latent variables, CFA, path analysis, and measurement invariance. You will learn how to specify, estimate, and test SEM models in R using the lavaan package, as well as perform power analysis through simulation.
